Privacy Policy

Last updated: 2025-11-04

KODE X – Institutional Trading AI connects to a private backend API (FastAPI on Railway) and to OpenAI’s API to process user prompts and generate responses.

1. Data We Process

  • Text prompts, parameters (e.g., pair, timeframe, risk settings) that you send to our API endpoints (/ai, /analyze).
  • Technical metadata for reliability: timestamps, latency, and minimal logs for error diagnostics. No persistent personal identifiers are stored intentionally.

2. Purpose of Processing

  • To generate model outputs (analysis, recommendations, summaries) via OpenAI.
  • To improve reliability and troubleshoot errors.

3. Retention

We do not intentionally store prompts or outputs long-term on our servers. Transient logs may exist for short periods for reliability and security.

4. Third Parties

We use OpenAI to process text and generate responses. Your requests may be transmitted to OpenAI’s API under our API key and subject to OpenAI’s policies.

5. Security

  • Transport encryption (HTTPS) is enforced.
  • Service access requires an API key header (X-API-KEY) for protected routes.

6. Your Choices

  • Avoid sharing personal or sensitive data in prompts.
  • Contact us to request deletion of specific diagnostic logs if any exist.
